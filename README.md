# VineVideo

This iOS application is sample app for creating Vine style video and written in swift.

## How to use

1. Clone project
1. Build app and run
1. Tap 'start' button
1. Tap 'pause' and 'resume' button
1. Tap 'stop' button
1. You'll find Vine style video in your library
